ZhakraPro
version: 1.1.0

Zhakrapro version 1.1.0 is a digital forensics GUI app written in Python 2.7.14+ and PyQt4.
It is a file integrity validator, system and device internet monitor. 
ZhakraPro 1.1.0 was built and tested on both 32/64bits linux machines.
The updated version now includes internet metrics, system storage evaluation and the ability to track geolocation of established connections.

Internet Monitor:
1. View device established connections
2. View running processes within device
3. View running processes and users of those processes
4. View network statistics.
5. View tcp metrics
6. View storage space
7. IP/Website geolocation tracker.

File Validation:
1. Enter File path
2. Select a Hash Function

Using zhakrapro1-1 in its simplest form:
unzip zhakrapro1-1-linux-app-dist-x86_64.zip as the case maybe and scroll down to zhakrapro1-1 and double click.
Viola!
